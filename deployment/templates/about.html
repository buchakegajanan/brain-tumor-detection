About This Project

Project Overview

This is an end-to-end deep learning project for automated brain tumor detection from MRI images. It uses state-of-the-art CNN architectures and transfer learning to achieve high accuracy in tumor classification.

Technology Stack

  • Deep Learning: TensorFlow, Keras
  • Models: Custom CNN, MobileNetV2, ResNet50
  • Explainability: Grad-CAM
  • Web Framework: Flask
  • Database: SQLite/MySQL
  • Frontend: Bootstrap 5, HTML, CSS, JavaScript

Features

  • User authentication and registration
  • MRI image upload and prediction
  • Grad-CAM visualization for explainability
  • Prediction history tracking
  • Confidence score display
Disclaimer

This is an educational project for academic purposes only. It should not be used for actual medical diagnosis or treatment decisions.
